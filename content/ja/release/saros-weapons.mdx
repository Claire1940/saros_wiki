惑星カルコサの過酷な環境を生き抜くには、単なる反射神経以上のものが必要です。それは、自由に使える様々な**Sarosの武器**を根本から理解することです。スルタリ・エンフォーサーであるアルジュン・デヴラジとして、プレイヤーは環境そのものがアーマーを汚染し生存を脅かす世界を探索しなければなりません。待ち受ける宇宙的恐怖を打ち破るには、標準的な銃器と高出力のパワーウェポンのニュアンスをマスターすることが、ミッションを成功させる鍵となります。このガイドでは、これまでに公開されたすべての武器タイプについて、発射モード、アップグレードパス、そしてゲーム独自の「汚染」メカニズムとの相互作用を含め、包括的に解説します。伝統的な副兵装を扱う場合でも、自らの腕を通じて星の力を操る場合でも、**Sarosの武器**システムを理解することがカルコサ奪還への近道です。

## スルタリ軍標準支給銃器

アルジュンの攻撃能力の基盤となるのは、標準的なスルタリ支給装備です。これらの武器は信頼性が高く、汎用性に優れており、さまざまな戦闘シナリオに適応できる独特の発射モードを備えています。後に発見される特殊なエネルギー武器とは異なり、これらの銃器は機械的なリロードや伝統的な弾道など、より手応えのある操作感が特徴です。

### スルタリ・エンフォーサー・ピストル
このピストルはアルジュンのデフォルトの副兵装として機能します。重火器に比べると地味に見えるかもしれませんが、2つの異なる弾丸挙動を備えています。一部の遭遇戦では、ピストルは型破りで広範囲な軌道を描く紫色のエネルギーを放ちます。これは群がる小型の敵を処理するのに最適です。精密射撃が必要な場面では、伝統的な直線軌道の白い単発弾に切り替わり、遠くのターゲットを正確に射抜くことができます。

### タクティカル・ショットガン
Sarosのショットガンは、Housemarkの武器デザインの真骨頂です。射撃のたびに次の装弾へと回転する、目に見える回転式バレル・アセンブリが特徴です。ゲームにおけるショットガンの多くは近接戦闘に限定されていますが、**Sarosの武器**バージョンのショットガンは驚くほど長い有効射程を誇ります。

| 特徴 | 詳細 |
| :--- | :--- |
| **基本装弾数** | 5発（6発までアップグレード可能） |
| **メイン射撃** | 中距離でも有効な高威力バックショット |
| **オルトモード** | 範囲ダメージを与える放物線状の迫撃砲ショット |
| **視覚的指標** | 精密な迫撃砲設置のための赤い弾道アーク |

### 多目的アサルトライフル
アサルトライフルは、スルタリ・エンフォーサーの主力武器です。過去のHousemark作品に見られた機能を引き継ぎつつ、マニュアル操作の深みが加わっています。プレイヤーは主に2つの発射モードを使い分けることになります：
1.  **標準弾:** 安定したDPS（秒間ダメージ）を出すための、直線的で高速な単発射撃。
2.  **デュアル・ホーミングショット:** 外側に向かって放たれた2つの弾丸が、ターゲットに収束するように追尾します。

> **💡 ヒント:** 部分的な遮蔽物の後ろに隠れている敵と戦うときは、デュアル・ホーミングショットを使用しましょう。外側への弧を描く軌道により、正面の盾をバイパスできることがよくあります。

## プラズマ・ダブルクロスボウ

アーセナルの中で最もユニークな追加要素の一つが、プラズマ・ダブルクロスボウです。この武器には伝統的な弾薬数の概念がありません。代わりに、HUD上の水平な「クールダウン」バーを使用します。発射するとバーが左から右に溜まり、バーがリセットされるまで再発射できませんが、発射モードによっては連射速度を上げることが可能です。

### 発射モード比較

| モード | エネルギーの色 | 発射数 | 挙動 |
| :--- | :--- | :--- | :--- |
| **トライビーム** | 紫 | 3 | 高ダメージ、広い水平拡散 |
| **ラピッドシングル** | オレンジ/白 | 1 | 低クールダウン、高頻度の連射 |
| **5連ビームアレイ** | 白/青 | 5 | 飛行する敵の群れを掃討するための最大拡散 |

ダブルクロスボウはカルコサの翼を持つ怪物に対して特に有効です。5連ビームモードの広い水平配列は、機敏な敵であっても回避することをほぼ不可能にします。

## パワーウェポンと汚染メカニズム

『Saros』では、過去のHousemarkゲームの「オルトファイア（特殊射撃）」システムが「パワーウェポン」システムへと進化しました。これらは単なる副次的な攻撃ではなく、アルジュンのアーマーの完全性に直結した生存のための不可欠なツールです。

### 汚染の浄化
「エクリプス（蝕）」の間、カルコサの世界は汚染されます。敵は赤い弾丸を放ち、それが命中すると体力を削るだけでなく、アーマーバーを「汚染」します。これはヘルスメーターの一部が赤く光ることで視覚化され、その部分は体力を最大まで回復できなくなります。

**パワーウェポンを発射することが、この汚染を浄化する主な方法です。** これらの高エネルギー攻撃で敵を倒すことにより、HUDから赤い汚染が消去され、アルジュンはアーマーの最大耐久値を復元できるようになります。

### 判明しているパワーウェポンの種類
パワーウェポンは、惑星での最初の死の後に物理的な変貌を遂げたと思われるアルジュンの右腕から放たれます。

1.  **スタンダードブラスト:** 広範囲に甚大なダメージを与える、オレンジ色のエネルギーによる壊滅的な一撃。
2.  **精密ビーム:** 照準を覗き込む必要があり、HUD上の2つのプログレスバーをチャージします。チャージが完了すると、複数の敵を貫通する星のような白い連続ビームを放ちます。
3.  **エネルギーSMG:** 「ホロウシーカー」に似た機能を持ち、ターゲットを追尾する青いエネルギーボルトを高速連射します。

> **⚠️ 警告:** パワーウェポンにはHUDの薄い青色のバーで示される制限容量があります。この容量は、アーマーマトリックス・アップグレードの「コマンド（Command）」ブランチに関連しています。

## オーバードライブ：究極の攻撃

状況が絶望的になったとき、アルジュンは**オーバードライブ**を発動できます。これは**L3 + R3**を同時に押し込むことで発動する「アルティメット」アビリティです。発動すると、アルジュンは上体を反らし、胸から巨大な赤いエネルギーを放出します。

| アビリティ要素 | 説明 |
| :--- | :--- |
| **発動方法** | L3 + R3 (スティック押し込み) |
| **リソース** | アーマー耐久値バーの下にある白いメーター |
| **効果** | 巨大な範囲ダメージと一時的な無敵時間（無敵フレーム） |
| **チャージ方法** | 敵の撃破、または時間の経過による自然回復 |

これらのシステムの開発に関する詳細については、[Housemark公式サイト](https://housemark.com/)を訪れ、3D空間における「弾幕（Bullet Hell）」戦闘の設計思想を確認してください。

## Sarosの武器における高度な戦闘戦略

Sarosで秀でるためには、世界の現在の状態に基づいて装備をローテーションさせることを学ばなければなりません。通常の探索フェーズで使用する武器と、エクリプス中に頼るべき武器は同じであってはいけません。

### HUDの管理
常に画面左下の隅に注意を払ってください。湾曲したバーはアーマーの耐久値（Armor Integrity）を表しています。そのバーに赤色が混じり始めたら、メインのアサルトライフルの使用を止め、すぐにパワーウェポンに切り替えてください。たとえ敵の体力が残り少なくても、パワーウェポンでとどめを刺すことが、次の戦闘で不利な状況（最大体力の減少）に陥らないための唯一の方法です。

### アーマーマトリックスのアップグレード
**Sarosの武器**の有効性は、アーマーマトリックスと密接に関係しています。特にパワーウェポンを多用するプレイスタイルを好むなら、**コマンド・ブランチ（Command Branch）**を優先すべきです。
*   **ティア1:** シールド強度の上昇。
*   **ティア2:** パワーウェポンのエネルギー容量拡大。
*   **ティア3:** オーバードライブのクールダウン短縮。

## よくある質問（FAQ）

**Q: Sarosでは複数のメイン武器を持ち運べますか？**

A: いいえ、アルジュンは通常、1つのメイン銃器（ショットガンやアサルトライフルなど）と、腕に装備された1つのパワーウェポンを携行します。メイン武器は環境内で新しいものを見つけることで交換できます。

**Q: ショットガンの弾薬容量を増やすにはどうすればいいですか？**

A: 弾薬容量は、カルコサ各地で見つかる武器固有のアップグレードを通じて増加します。例えば、ショットガンは最初5発ですが、「大容量シリンダー」を拾うことで6発にアップグレードできます。

**Q: ダブルクロスボウには弾薬のピックアップが必要ですか？**

A: いいえ、ダブルクロスボウは熱/クールダウンメカニズムを使用します。弾数は無限ですが、現在の発射モードの頻度に応じて、水平のプログレスバーがリセットされるまで待ってから再発射する必要があります。

**Q: 汚染を素早く浄化するのに最適な武器は何ですか？**

A: 一般的に、SMGスタイルのパワーウェポンが汚染浄化に最適です。汚染は敵を倒すごとに「少しずつ」浄化されるため、強力な敵に単発ブラストを撃ち込むよりも、SMGモードで弱い敵の群れを一掃する方が効率的だからです。
