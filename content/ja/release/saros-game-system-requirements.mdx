## Saros 最小動作環境

1080p解像度、低〜中設定で安定した30〜60 FPSを目指すプレイヤーにとって、最小要件は現代のミドルレンジ構成であれば比較的達成しやすいものです。しかし、Sarosは弾道システムや環境破壊に高度な物理演算を使用しているため、アセットの読み込みによるカクつきを防ぐためにソリッドステートドライブ（SSD）が必須となります。

| コンポーネント | 最小スペック |
| :--- | :--- |
| **OS** | Windows 10/11 (64-bit) |
| **プロセッサ** | Intel Core i5-11600K / AMD Ryzen 5 5600X |
| **メモリ** | 16 GB RAM |
| **グラフィックス** | NVIDIA GeForce RTX 3060 / AMD Radeon RX 6600 XT |
| **DirectX** | Version 12 |
| **ストレージ** | 80 GB の空き容量 (**SSD必須**) |

> **警告：** 従来のハードディスクドライブ（HDD）でSarosを実行すると、激しいアクションシーン中に大幅なロード遅延や音声のズレが発生する可能性が高いです。

## ゲームプレイメカニクスがパフォーマンスに与える影響

Sarosのゲームプレイは「軽快で満足感がある」と評されていますが、そのスピード感はハードウェアへの負荷と引き換えです。本作は敵が色分けされた弾丸を放つ3D弾幕システムを採用しています。黄色の爆発にはダッシュ（R1）が必要で、青色のものはシールド（L1）で吸収しなければなりません。

画面上に数十もの弾丸が同時に存在する際、ゲームエンジンは1秒間に数千回の当たり判定（コリジョンチェック）を行います。そのため、**Sarosの動作環境** は特にCPU負荷が高くなります。プロセッサが古い場合、グラフィックボードが最高級であっても「マイクロスタッタリング（微細なカクつき）」が発生することがあります。

### 日食（ソーラー・エクリプス）モディファイア
Sarosの最もユニークな機能の一つは、日食を発生させる能力です。このメカニクスは動的に環境を変化させ、敵の凶暴性と戦利品の質を高めます。技術的な観点からは、画面上のアクティブな光源とパーティクルエミッターの数が増加します。

| 機能 | ハードウェアへの影響 | 最適化のヒント |
| :--- | :--- | :--- |
| **日食** | GPU負荷：高 | 影の品質を「中」に下げる |
| **弾幕パーティクル** | CPU負荷：高 | バックグラウンドアプリを閉じる |
| **3Dオーディオログ** | 負荷：低 | 没入感を高めるために空間オーディオを使用 |
| **高速移動** | SSDスループット：高 | NVMe Gen 4 または Gen 5 ドライブを使用 |

## グラフィック設定と最適化

PCで安定したフレームレートを維持するのに苦労している場合は、調整すべき主要な設定がいくつかあります。Sarosは最新のアップスケーリング技術をサポートしており、異星の巨大なアリーナで高いリフレッシュレートを実現するために不可欠です。

1. **アップスケーリング技術**: DLSS 3.5またはFSR 3.0を有効にしてください。これらは視覚的な鮮明さをほとんど損なうことなく、パフォーマンスを大幅に向上させます。
2. **パーティクルの品質**: Housemarqueのゲームはパーティクル演出で知られています。これを「ウルトラ」から「高」に下げるだけで、ボス戦中のGPU負荷を最大15%削減できます。
3. **容積測定フォグ（Volumetric Fog）**: 異星の雰囲気には濃い霧が使われています。この霧の解像度を下げることで、湿地のようなバイオームでのFPSが安定します。
4. **反射**: 拠点の濡れた路面でのレイトレーシングは見事ですが、スクリーンスペースリフレクション（SSR）に切り替えるのが、20〜30 FPSを獲得する最も手っ取り早い方法です。

> **💡 ヒント:** ノートPCでプレイしている場合は、電源プロファイルが「ターボ」または「最高パフォーマンス」に設定されていることを確認してください。Sarosはシステムが熱を感知すると、すぐにスロットリング（性能制限）が発生します。

## 永続的な成長要素とハードウェアの安定性

Sarosには複雑なスキルツリーによる永続的な成長システムがあるため、順調なランの最中にクラッシュすることだけは避けたいものです。「ドライブ」「コマンド」「バイタリティ」といったステータスにリソースを投入して強化できますが、システムがクラッシュすると、一時的な通貨やアーティファクトを失う可能性があります。

ゲームの構造は、死後にキャンプに戻る「ループ」を中心に構築されています。これらの遷移はPS5 Proではシームレスですが、PC版で物語の「無駄のない」テンポを維持するには、高速なSSDが必要不可欠です。

## 2026年に32GBのRAMが新標準となる理由

**Sarosの動作環境** の最小スペックには16GBと記載されていますが、カクつきのない体験のためには32GBを強く推奨します。Sarosはバイオームのランダム構造を処理するために、大量のレベルデータをメモリに保持します。ストーリー上の理由やスキルツリーへのポイント投入のために拠点にテレポートする際、十分なRAMがあればドライブからアセットを再取得する必要がなくなり、初期のレビュアーが指摘していた「時折発生する小さなカクつき」を防ぐことができます。

## FAQ

**Q: SarosをSteam Deckや携帯型PCで動作させることはできますか？**

A: **Sarosの動作環境** はかなり高いですが、Steam Deck OLEDやROG Allyなどのハイエンド携帯機であれば、「低」設定と積極的なFSRアップスケーリングを使用することでプレイ可能です。ほとんどのエリアで30〜40 FPSを期待できますが、ボス戦では低下する可能性があります。

**Q: Sarosはウルトラワイドモニターに対応していますか？**

A: はい、Sarosは21:9および32:9のアスペクト比をフルサポートしています。広い視野によって異星のスケール感が大幅に強調され、周囲から飛んでくる弾丸をより簡単に察知できるようになります。

**Q: 設定で「激ムズ」な難易度を下げる方法はありますか？**

A: 伝統的な「イージーモード」はありませんが、ワールドに入る前にオーバークロックシステムを使用して、体力増加や武器ダメージアップなどのモディファイアを自分に付与できます。ただし、これらには通常、汚染ダメージの増加などの代償が伴うことに注意してください。

**Q: なぜ日食中に銃がジャム（詰まる）のですか？**

A: これはバグではなくゲームプレイの仕様です。日食中の特定の戦利品にはトレードオフがあります。例えば、エネルギー生成効率が上がる代わりに、ダメージを受けると銃がジャムるようになるといったものです。ポーズメニューで現在のアクティブなアーティファクトとモディファイアを確認し、ペナルティを把握してください。
