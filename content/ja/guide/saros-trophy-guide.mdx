ハイテンションなローグライクゲームでプラチナトロフィーを獲得するには、単なるスキル以上のものが必要です。探索と収集に対する体系的なアプローチが求められます。**Saros トロフィーガイド**をマスターすることは、次の大型コンテンツアップデート前に100%コンプリートを目指すすべてのスカウトにとっての第一歩です。このゲームは戦闘技術と細かな環境探索の両方に報酬を与えるため、この包括的な**Saros トロフィーガイド**は、危険なバイオームをナビゲートし、見逃した収集品を特定し、ランを最適化して数十時間を節約するのに役立ちます。

プラチナへの道は、主に各バイオームが提供するほぼすべてを見て、実行する必要がある「調査」トロフィーによって制限されています。標準的なアクションゲームとは異なり、マップが自動生成される性質上、一部のトロフィーはRNG（乱数生成）に左右されます。以下に概説する戦略に従うことで、これらの確率を操作し、外科的な精度でコレクションを完成させることができます。

## Saros トロフィーガイドの核心メカニクス

プラチナを獲得するには、バイオーム調査の構成要素を理解する必要があります。6つのバイオームにはそれぞれ、関連するトロフィーをアンロックするために完了しなければならない特定のチェックリストがあります。エリアのボスを倒すことは最も明白な要件ですが、本当の挑戦は隠されたゼノグリフ・サイファーとスカウトログを見つけることにあります。

| 要件 | 説明 | マップアイコン/インジケーター |
| :--- | :--- | :--- |
| **バイオームボスの撃破** | エリアの主要な守護者を排除する。 | 赤いボスの扉 |
| **コンテインメント・ルーム** | チャレンジルーム内の敵の全ウェーブをクリアする。 | **金の扉** |
| **シークレット・ルーム** | 隠されたサブレベルまたはテレポーター・ルームにアクセスする。 | 光る床 / オーブ |
| **ゼノアーカイブ** | すべてのホログラフィック歴史ディスプレイにインタラクトする。 | 青いパーティクルの部屋 |
| **スカウトログ** | 以前のスカウトが残したすべての音声記録を収集する。 | 白い信号アイコン |
| **ゼノグリフ・サイファー** | 壁にあるすべての光る赤いルーンを見つける。 | 赤い三角形アイコン |

> **💡 ヒント:** 永続的な移動アップグレード（グラップリングフックやデルフィック・バイザーなど）をすべてアンロックするまで、完全な調査完了を試みないでください。それらがないと物理的にアクセスできない収集品が多く存在します。

## 効率的な調査完了戦略

多くのプレイヤーはトロフィー獲得の際、すべての部屋をクリアし、すべての敵と戦うという間違いを犯します。プラチナが目標なら、「スピードラン」モードに切り替える必要があります。**Saros トロフィーガイド**の戦略に従うということは、足りない収集品に関係のない障害物を無視することを意味します。

どのスカウトログやサイファーが足りないかを特定したら、メインパスを駆け抜け、不足しているアイテムが出現する可能性のあるサイドルームだけに立ち寄るべきです。必要な部屋が出現しなかった場合は、すぐにサイクルをリスタートしてください。この「サイクル・リセット」法は、進展のないランを最後まで終えるよりも大幅に高速です。

### バイオーム 1 & 2：基礎
初期のバイオームでは、一部の収集品が特定のメカニクスに関連付けられています。例えばバイオーム1では、再構成装置（リコンストラクター）を初めて使用した*後に*のみ、その部屋にスカウトログが出現します。同様に、パラサイトが導入される部屋では、最初のパラサイトを収集した後の訪問時にログが出現します。

バイオーム2（紅蓮の荒野）では、山の構造が砂漠の麓、下層、山頂に分かれています。上級プレイヤーはこの**Saros トロフィーガイド**の戦術を使用します。麓から山頂へ直接ポータルで移動し、そこから山を下るように逆走します。これにより、手動で登るよりもはるかに速く、サイファーの出現確率が高い場所をチェックできます。

## 進捗の追跡

プラチナ狩りで最も難しいのは、どの特定の部屋を探しているのかが分からないことです。すべての**Saros トロフィーガイド**は、ゲーム内のデータバンクを使用して収集したログを照合することの重要性を強調しています。

| バイオーム | 見逃しやすいもの | 場所のヒント |
| :--- | :--- | :--- |
| **バイオーム 1** | スカウトログ 09 | **施錠された青い扉**の向こう（鍵が必要） |
| **バイオーム 3** | 橋のサイファー | 橋を架ける*前*にのみアクセス可能。 |
| **バイオーム 4** | 太陽の顔の断片 | 植物のような構造物の近くにあるサイドパス内。 |
| **バイオーム 5** | コンテインメント・ログ | 3つの主要な扉があるハブ・ルームに位置。 |
| **バイオーム 6** | 深海のサイファー | 最後の降下中の破壊可能な壁の後ろに隠されていることが多い。 |

## 高度なナビゲーションのコツ

**Saros トロフィーガイド**では、武器の習熟度が十分になったら、オプションの戦闘はすべてスキップすることを推奨しています。例えばバイオーム5の部屋は非常に難易度が高いことで知られています。もし「凍結」セクションのスカウトログだけが足りない場合は、セバード（断絶されし者）を走り抜け、部屋のパターンを確認し、必要でない場合はそのまま深淵（バイオーム6）へ飛び込みましょう。

### RNG（運要素）への対処
一部の部屋は出現率が「レア」で、10回に1回程度しか出現しないことがあります。効率を最大化するには：
1. **オフラインでプレイ:** 一部のプレイヤーは、オフラインモードにすることで他プレイヤーの「死体」部屋が出現しなくなり、結果としてレアな収集品部屋の出現率が上がる可能性があると報告しています。
2. **マップを確認:** サイファーはミニマップ上に小さな赤い三角形として表示されます。壁のルーンを視認する必要はありません。部屋を駆け抜けながらマップをちらっと見るだけで十分です。
3. **ショートカットを利用:** 各バイオームには次のエリアへのショートカットがあります。ボス関連のトロフィーを獲得したら、これらを使ってボス戦をスキップしましょう。

## 最終クリーンアップ：太陽の顔の断片

隠しエンディングと最終トロフィーを完了するには、太陽の顔の断片を見つける必要があります。各バイオームに1つずつあり、ゲームを一度クリアしてすべての「家」のシーケンスを見た*後に*のみ出現します。

1. **バイオーム 1:** 開始地点近くの、グラップリングフックが必要な隙間を探す。
2. **バイオーム 2:** 砂漠エリアにあり、多くの場合、水上歩行アップグレードが必要。
3. **バイオーム 3:** 橋のセクションの終わり近くにある。
4. **バイオーム 4:** 残響の遺跡の開始地点近くのサイドルームをチェック。
5. **バイオーム 5:** ヴォルト・ルーム（金庫室）内に位置（戦闘不要）。
6. **バイオーム 6:** 光るオーブを撃ったときにのみ現れるプラットフォームを探す。

特定の敵との遭遇に関する詳細な戦略については、エキスパートコミュニティのアドバイスが掲載されている[公式PlayStationトロフィーポータル](https://www.playstation.com/en-us/support/games/playstation-trophies-guide/)をご覧ください。

## FAQ

**Q: Saros トロフィーガイドにおいて、取り返しのつかない（見逃しやすい）トロフィーはありますか？**

A: サイクルに基づいたローグライクゲームであるため、厳密に取り返しのつかないトロフィーはありません。ただし、太陽の顔の断片のように特定のストーリー展開後にしか出現しない収集品もあるため、本編クリア後に戻る必要がある場合があります。

**Q: 調査トロフィーを獲得するために、毎回ボスを倒す必要がありますか？**

A: いいえ。一度ボスを倒せば、調査トロフィーのために再度戦う必要はありません。環境内の残りのサイファーやスカウトログを探すことに専念できます。

**Q: すべて見つけたはずなのに、調査が95%で止まっているのはなぜですか？**

A: ゼノアーカイブを再確認してください。これらはホログラムの部屋です。調査にカウントされるには、部屋にあるすべてのホログラムにインタラクトする必要があります。また、その特定のバイオームで少なくとも1つの「シークレット・ルーム」（落とし戸や光るオーブの部屋）に入ったことを確認してください。

**Q: 足りないサイファーを追跡できますか？**

A: ゲーム内の「リソース」タブで、各バイオームの合計に対して何個のサイファーを見つけたかを確認できます。ただし、どの特定のサイファーが足りないかは教えてくれません。視覚的なリファレンスとマップのレイアウトを比較して絞り込む必要があります。
