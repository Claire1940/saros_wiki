## Comment démarrer le préchargement de Saros sur PS5 et PSVR 2

Lancer votre téléchargement est un processus simple, mais il y a quelques prérequis. Tout d'abord, vous devez avoir une précommande numérique confirmée via le PlayStation Store. Si vous avez acheté un disque physique, vous ne pourrez pas participer au préchargement numérique et devrez attendre l'arrivée du disque.

Suivez ces étapes pour vous assurer que votre **préchargement de Saros** démarre correctement :

1.  **Accédez à votre bibliothèque :** Allez dans l'onglet « Collection » sur l'écran d'accueil de votre PS5.
2.  **Sélectionnez Saros :** Trouvez l'icône du jeu dans votre liste d'achats.
3.  **Vérifiez le compte à rebours :** Vous verrez un minuteur décompter le temps jusqu'au lancement. Cliquez sur le minuteur pour afficher les paramètres de « Téléchargement automatique ».
4.  **Activez le téléchargement automatique :** Assurez-vous que la case pour le jeu et tout DLC disponible (comme le Sol Tari Shield ou les bonus de précommande) est cochée.
5.  **Déclenchement manuel :** Si le téléchargement ne démarre pas automatiquement le 28 avril, cliquez manuellement sur le bouton « Télécharger » depuis cet écran.

## FAQ

**Q : Puis-je jouer à Saros en avance si je termine le préchargement de Saros plus tôt ?**

R : Non, le préchargement ne fait que télécharger les fichiers sur votre système. Le jeu reste « verrouillé » derrière un minuteur numérique qui expire à minuit le 30 avril 2026.

**Q : Le préchargement de Saros inclut-il le patch Day One ?**

R : Dans la plupart des cas, oui. Le système de Sony regroupe généralement le dernier patch disponible avec les fichiers de téléchargement initiaux, bien qu'un petit correctif (« hotfix ») puisse encore se déclencher le jour du lancement effectif.

**Q : Le bouclier Sol Tari est-il disponible pour les joueurs de l'Édition Standard ?**

R : Le bouclier Sol Tari est une mécanique de jeu disponible pour tout le monde, mais la précommande ou l'achat de l'Édition Deluxe peut vous octroyer une apparence unique ou un accès plus précoce à des améliorations spécifiques pour celui-ci.

**Q : Que se passe-t-il si mon téléchargement est interrompu ?**

R : Le système PS5 est conçu pour reprendre les téléchargements là où ils s'étaient arrêtés. Vous n'avez pas besoin de recommencer tout le processus de plus de 115 Go si votre connexion internet vacille ou si vous devez mettre le téléchargement en pause.
