4.  Cliquez sur **Confirmer** pour lier le jeu à votre profil.

## FAQ

**Q : Puis-je utiliser des clés CD Saros d'une autre région ?**

R : Généralement, non. La plupart des clés sont verrouillées par région pour empêcher le commerce sur le "marché gris" entre différentes zones économiques. Si vous achetez une clé destinée à l'Europe et essayez de l'activer en Amérique du Nord, la plateforme bloquera probablement la transaction. Recherchez toujours des clés "Global" si vous n'êtes pas sûr des restrictions de votre région.

**Q : Que dois-je faire si mes clés CD saros arrivent sous forme d'image scannée ?**

R : C'est une pratique courante chez certains revendeurs qui s'approvisionnent en stocks physiques et numérisent les codes. Tant que le code est lisible et que le vendeur est réputé, c'est parfaitement sûr. Cependant, si l'image semble altérée ou si le code est masqué, contactez immédiatement le service client du vendeur avant de tenter de l'utiliser.

**Q : Ces clés expirent-elles si je ne les utilise pas tout de suite ?**

R : Les clés de jeu standard n'ont généralement pas de date d'expiration. Cependant, les clés promotionnelles (telles que celles provenant d'un bundle matériel ou d'un cadeau à durée limitée) peuvent avoir une date limite d'utilisation. Il est toujours préférable d'activer votre clé dès que vous la recevez pour éviter tout problème potentiel de mise à jour de base de données ou d'erreur du vendeur.

**Q : Est-il possible de se faire rembourser une clé CD après qu'elle a été révélée ?**

R : La plupart des places de marché tierces ont une politique stricte de "non-remboursement" une fois que le code numérique a été affiché à l'utilisateur. En effet, la clé est alors considérée comme "consommée" car le vendeur ne peut pas vérifier si vous avez enregistré le code pour une utilisation ultérieure. N'achetez auprès d'un vendeur que si vous êtes certain à 100 % de vouloir le produit.
