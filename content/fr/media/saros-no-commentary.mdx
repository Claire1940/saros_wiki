## Maîtriser l'atmosphère de Carcosa

La planète Carcosa n'est pas un environnement statique. C'est un cimetière vivant et changeant d'une ancienne civilisation qui exige une vigilance constante. Au fil de votre exploration, vous remarquerez que les données cartographiques deviennent fréquemment corrompues ou non enregistrées, vous obligeant à vous fier aux points de repère visuels et au suivi du signal de votre équipement Echelon.

L'élément le plus dangereux de l'environnement est l'Éclipse. Durant ces événements, le rendement énergétique du monde est faussement enrichi, mais cela déclenche également des crises psychologiques chez les survivants. Gérer vos niveaux de « Chaleur » et rester attentif aux distorsions spatiales sont les clés pour atteindre les structures des temples profonds et les systèmes de tunnels souterrains.

> **Attention :** Les distorsions spatiales peuvent offrir des raccourcis, mais elles vous déposent souvent dans des zones de haute menace sans avertissement. Assurez-vous toujours que vos armes Sultari sont chargées avant de pénétrer dans une distorsion.

## Naviguer dans le réseau géo ancien

Pour progresser dans la campagne principale, vous devez interagir avec le Géo-Réseau. Il s'agit d'une série de nœuds interconnectés liés à des hubs centraux qui contrôlent les foreuses massives et les systèmes de portes de la planète. L'accès à ces nœuds est souvent une question d'analyse entre risques élevés et profits importants.

### Étapes pour activer un nœud de réseau :
1. **Localiser le Hub :** Suivez le signal Sultari à travers les tunnels souterrains.
2. **Nettoyer la zone :** Les entités hostiles entrent souvent en mode « Réponse de sécurité » dès qu'on s'approche d'un nœud.
3. **Activer la console :** Utilisez vos modules d'intégration pour contourner le verrouillage.
4. **Se préparer au confinement :** Les portes peuvent se verrouiller derrière vous, vous obligeant à trouver un itinéraire alternatif ou un nœud de téléportation.

| Type de nœud | Fonction | Exigence |
| :--- | :--- | :--- |
| **Contrôle de porte** | Ouvre/Ferme les passages | Intégration standard |
| **Nœud de téléportation** | Voyage instantané entre secteurs | Calibrage tech antique |
| **Cache de l'Hollow** | Journaux d'histoire et fournitures | Code de contournement de sécurité |

## Stratégies de survie pour l'Éclipse

L'Éclipse est plus qu'un simple changement visuel ; c'est un événement qui modifie le gameplay. Quand le soleil « vous voit », comme le disent les survivants, le monde devient plus hostile. Vous entendrez le « murmure », et votre interface affichera probablement des avertissements concernant des besoins extrêmes en énergie cinétique.

Le visionnage des parties en **saros no commentary** révèle que les joueurs expérimentés utilisent souvent l'Éclipse pour collecter de la Lucenite enrichie, malgré le danger accru. Cette ressource est essentielle pour les améliorations de haut niveau, mais elle implique d'affronter des « entités déverrouillées » qui n'ont aucune donnée enregistrée dans la base de données de votre éclaireur.

> **Astuce :** Si vous entendez la « voix » pendant une éclipse, cherchez immédiatement refuge dans un tunnel. La mécanique de « rupture » psychologique peut rendre vos commandes inactives ou réinitialiser entièrement vos données cartographiques.
